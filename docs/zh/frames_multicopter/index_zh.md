# 多旋翼飞行器

PX4支持多种类型的单旋翼和多旋翼飞行器，包括直升机、三旋翼、四旋翼、六旋翼、十二旋翼等，具有多种不同的几何形状（+、宽型、x型等）。

::: tip
通用配置可以在[机架参考 > 多旋翼](../airframes/airframe_reference.md#copter)中查看。
这些配置可以在配置过程中进行自定义。
:::
