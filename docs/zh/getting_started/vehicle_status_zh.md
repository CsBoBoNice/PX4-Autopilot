# 飞行器状态通知

PX4 提供基于飞行器的视觉（LED）和声音（蜂鸣器）通知，用于"高级"飞行器状态和飞行准备情况。
这些通知指示，例如，飞行器是否正确校准、是否有 SD 卡、是否有位置锁定、是否可以安全接近、是否解锁、何时准备飞行等。

此外，PX4 在 QGroundControl 中提供更详细的飞行准备信息。
